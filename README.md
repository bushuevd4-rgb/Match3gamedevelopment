# 🍭 Сладкий Матч — Match-3 Game

> **с Яблочком и Кепа Митой**

Милая match-3 игра в пастельных тонах для русскоязычной аудитории с двумя харизматичными персонажами — добрым Яблочком и ворчливой Кепа Митой. Собирай сладости, выполняй цели уровней и открывай новые локации!

---

## 🎮 Геймплей

- **Поле:** сетка 8×8 с сладкими элементами — 🍎 яблоко, 🎂 торт, 🍩 пончик и другие
- **Цель:** набрать нужное количество очков за ограниченное число ходов
- **Механика:** меняй местами соседние элементы, чтобы собрать 3+ одинаковых в ряд или колонку
- **Комбо:** цепные совпадения дают бонусные очки и спецэффекты

## 👾 Персонажи

| Персонаж | Характер | Реплики |
|----------|----------|---------|
| 🍎 **Яблочко** | Добрая, поддерживающая | *«Ты справишься!»*, *«Вау! Это было восхитительно! 🌟»* |
| 😾 **Кепа Мита** | Язвительная, ворчливая | *«Опять промахнулся? Ну давай, соберись!»*, *«Комбо? Неожиданно...»* |

- Персонажи **реагируют на действия игрока** — говорят реплики каждые 6 ходов или при комбо
- Каждые **20 ходов Кепа Мита** взрывает случайный ряд
- Если **нет доступных ходов** — Яблочко магически перемешивает поле

## 🗺️ Карта уровней

Линейная карта с тремя локациями:

| Локация | Уровни |
|---------|--------|
| 🍎 Яблоневый сад | Ур. 1–10 |
| 🎂 Кондитерская | Ур. 11–20 |
| 👑 Сладкое королевство | Ур. 21–30 |

Уровни открываются последовательно. Пройденные уровни отмечаются галочкой ✓.

---

## 🕹️ Режимы игры

- **Классика** — набери цель за ограниченное число ходов
- **Гонка со временем** — максимум очков за 3 минуты
- **Выживание** — поле постепенно заполняется, продержись как можно дольше

---

## 🏆 Достижения и значки

- 🌩️ **«Гроза комбо»** — собери 10x комбо
- 🍬 **«Сладкоежка»** — собери все типы элементов за один ход
- ✨ **«Без единой ошибки»** — пройди уровень без использования подсказок
- 🔀 **«Мастер перемешивания»** — используй перемешивание 5 раз

Галерея достижений с анимацией разблокировки и возможностью установить значок на профиль.

## 🚀 Установка и запуск

### Требования

- [Node.js](https://nodejs.org/) версии **18+**
- npm (входит в состав Node.js)

### Шаги

**1. Клонируй репозиторий**
```bash
git clone https://github.com/bushuevd4-rgb/Match3gamedevelopment.git
cd Match3gamedevelopment
```

**2. Установи зависимости**
```bash
npm install
```

**3. Запусти сервер разработки**
```bash
npm run dev
```

**4. Открой в браузере**
```
http://localhost:5173
```

### Сборка для продакшена

```bash
npm run build
```

Готовые файлы появятся в папке `dist/`.

Для предпросмотра собранной версии:
```bash
npm run preview
```
